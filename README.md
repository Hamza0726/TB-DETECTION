  # TB Risk Detection Web App

  This is a code bundle for TB Risk Detection Web App. The original project is available at https://www.figma.com/design/ztdMhDywip8XHJf29uJkRO/TB-Risk-Detection-Web-App.

  ## Running the code

  Download the code in .zip format and extract it to your desired folder.

  Copy the folder 'path' where package.json exists.

  Open command prompt and run `cd [path you copied]`
  
  Run `npm install` to install the dependencies.

  Run `npm run dev` to start the development server.
